# Artifact and rights notice

This repository separates technical provenance from legal permission. Public
availability of a file does not by itself grant reuse, modification, or
redistribution rights.

## Source code

No open-source licence is currently declared for the application code. Add an
explicit `LICENSE` before describing the code as open source or granting reuse
rights.

## Audio stimuli

The four frozen top-level stimulus directories reproduce the files in the OSF
`OFFLINE AP TESTS.zip` distribution identified in `STIMULUS_MANIFEST.json`.
The versioned directories under `stimulus_sets/` are documented Praat-derived
reconstructions; each directory contains its own manifest and parameter record.

The upstream OSF project did not declare a licence in the locally audited
metadata. The repository owner has directed that the audio may be included in
this public repository, but that operational direction is not a representation
that upstream reuse rights have been granted to third parties. Downstream users
must determine whether their intended use is permitted and provide any required
attribution or permission.

## Articles and research data

Article PDFs and other local literature files are source materials for audit
only and are excluded from the public repository. Participant result files,
checkpoints, portal receipts, credentials, and directly identifying information
must never be committed.
